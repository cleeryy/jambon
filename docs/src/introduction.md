# Jambon

Jambon is a Rust Discord bot that provides a convenient interface to manage
Proxmox VE virtual environments directly from Discord.

## Features

- **VM Management** — List, start, stop, shutdown, and migrate virtual machines.
- **Container Management** — Manage LXC containers.
- **Node Monitoring** — View cluster node status, CPU, memory, and uptime.
- **Cluster Overview** — See all resources across your Proxmox cluster.
- **Slash Commands** — All commands are Discord slash commands with rich embeds.
- **Permission Control** — Role-based access via Discord permissions.
- **Background Monitoring** — Optional health alerts for node status changes.
